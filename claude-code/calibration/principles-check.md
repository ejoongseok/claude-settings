# 행동 지침 캘리브레이션 (18문항, 월 1회)

## 사용법

- 매월 첫 주, 새 세션에서 상황만 제시하고 Claude의 답을 정답과 대조한다 (예: "캘리브레이션 3번 상황이야, 어떻게 할래?")
- 불일치가 나오면 CLAUDE.md의 원리 문구를 고치는 게 아니라 분기 조건을 추가하는 방식으로 교정한다
- 정답 열은 2026-07-06 확정
- 채점 기록은 이 파일 하단에 날짜별로 append

## 관측 한계 (미발동 딱지)

- 이 문제집은 "원리를 아는가"를 채점하며, 실제 세션에서 "지켰는가"까지는 채점하지 못한다.
- 발동 여부가 로그에 남지 않는 원리: 문제 먼저(착수 전 확인), 복수안 전환(원리 4의 분기 발동), 산출물 검증 수행. 반기 점검에서 이 원리들의 발동 이력 0건은 "규칙 승리"가 아니라 "관측 불가"로 분류한다. 출석부 없는 개근상 금지.
- 보완 관측 수단: 세션 중 이 원리들이 실제 발동한 사례를 보면 세션 로그에 한 줄 남긴다(예: "원리 4 발동, 복수안 전환"). 강제 장치는 두지 않는다. 무거운 로깅 체계는 복잡도 비용이 편익을 넘어선다.
