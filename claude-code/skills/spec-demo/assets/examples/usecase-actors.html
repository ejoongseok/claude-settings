역할별 기능·동작 (유스케이스) — 예시

누가 어떤 기능을 쓰고, 같은 기능도 역할별로 어떻게 다른가

상황 — 역할별 기능·권한

각 역할이 무슨 기능을 쓰고, 같은 기능도 어떻게 다르게 동작하나요?

  • 역할(왼쪽)을 클릭 → 그 역할이 쓰는 기능 강조 ("누가 무엇을").
  • 기능(오른쪽)을 클릭 → 그 기능을 역할별로 어떻게 다르게 하는지 ("같은 기능, 다른 동작") — 예: 주문 조회를 작성자는 본인 것만, 관리자는 전체.
  • 권한 등급(조회·생성·관리)과 역할별 범위·제약을 같이 맞춥니다.
01

역할 ↔ 기능 ↔ 동작

역할 클릭 = 누가 무엇을 · 기능 클릭 = 역할별 동작 차이
역할이나 기능을 클릭해 보세요.
합의
이 역할-기능-동작이 맞나요? 작성자가 보는 범위(본인 것만? 부서 전체?), 결재자의 승인 범위(자기 차례만?), 관리자가 강제 승인·대리 등록까지 하는지 — 역할과 기능을 눌러 가며 맞춥니다.
참고
권한 등급(조회·생성·관리)은 일반적인 권한 축과 같습니다. 역할-기능 매핑과 역할별 동작은 대표 예시 — 실제 권한·범위와 맞춰야 합니다(합의 대상).
