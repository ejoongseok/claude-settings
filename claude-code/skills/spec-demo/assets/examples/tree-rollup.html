밀키트 구성 전개 (다단계 소요량) — 예시

완제품 → 구성팩 → 재료, 단계별 소요량

밀키트 생산 수량을 바꾸면 모든 단계의 재료 필요량이 자동으로 다시 계산됩니다.

  • 트리를 펼치면 밀키트가 어떤 구성팩으로, 그 구성팩이 또 어떤 재료로 구성되는지 단계별로 보입니다.
  • 위에서 생산 수량을 바꾸면 각 단계 필요량이 상위 수량 × 단위 소요 × 여유분으로 다시 계산됩니다.
  • 노드를 누르면 품목 코드·단위 소요·여유분·구매/외주 구분을 봅니다.
