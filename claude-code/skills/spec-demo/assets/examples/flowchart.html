상황 — 결재 승인 흐름

결재는 어디서 승인·반려로 갈라지고, 다음 결재자로 되돌아 도나요?

  • 주 경로는 세로, 분기는 옆으로 갈라집니다 — 검토에서 반려는 오른쪽 종료로, 다음 결재자 "아니오"는 완료로.
  • 판단 노드(노란색)에서 승인/반려, 예/아니오를 선택하면 그 경로를 따라가고, "예"는 검토로 되돌아가는 루프를 직접 돌려볼 수 있습니다.
  • 이 분기·루프가 실제 결재 업무와 맞는지 같이 맞춥니다.

참고
이건 업무 프로세스 흐름(판단·분기)입니다. 데이터 상태 전이는 상태도로 따로 봅니다. 결재 세부 흐름은 대표 흐름 가정(합의 대상).
