앞으로 진행, 뒤로 되돌리기 — 어디로 갈 수 있나

상황 — 주문 진행과 되돌리기

주문은 앞으로 진행하고, 필요하면 되돌릴 수 있어야 하나요?

  • 현재 상태에서 진행 전이(파란 버튼)와 되돌리기 전이(점선 버튼)가 나뉘어 보입니다 — 결제 취소, 출고 보류처럼 뒤로 가는 경로.
  • 버튼을 누르면 상태가 이동하고 거쳐온 경로가 남습니다. "완료에서 배송 취소로 되돌릴 수 있나?"가 한눈에.
  • 특정 상태로 바로 가려면 위 상태 노드를 직접 클릭하세요(예시 — 한 번에 이동). 아래 버튼은 규칙상 허용되는 전이입니다.
  • 어떤 되돌리기가 허용되는지, 그 트리거가 무엇인지가 핵심 합의입니다.

합의
어떤 되돌리기가 허용되나요? 완료(CO)에서 배송 취소로 배송대기 복귀가 가능한지, 준비중(PR)을 결제완료(PD)로 보류할 수 있는지, 되돌릴 때 트리거(이벤트·권한)는 무엇인지 — 진행/되돌리기 버튼을 눌러 보며 맞춥니다.
참고
진행 전이 트리거(결제·출고 지시·배송)는 실제 규칙 기준으로 두고, 역전이(되돌리기)의 허용 여부와 트리거는 대체로 가정(합의 대상)으로 표시합니다.
